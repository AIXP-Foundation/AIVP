# Changelog

All notable changes to the AIVP protocol will be documented in this file.

## [1.0.0-draft] - 2026-03-26

### Added
- Initial protocol specification (Part I-IV)
- 5 Architecture Decision Records (ADR-001 through ADR-005)
- 18 message types across 6 categories
- AIVP Trust model (V0-V4) with Sybil resistance
- USD denomination with multi-stablecoin settlement
- Logic Vault escrow with milestone-gated release
- AgentSLA quality metrics (aligned with arXiv 2511.02885)
- Dispute resolution (4-tier: optimistic, direct, arbitrator, Kleros)
- Depeg circuit breaker for settlement stability
- Email transport with Ed25519 signatures and DMARC enforcement
- Optional AIVP ID (aivp_id) commercial identity system
- 2 example interaction flows (simple-payment, dispute-resolution)
- README in English and Chinese
