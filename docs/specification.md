# AIVP Specification Overview

## Reading Guide

The AIVP specification is organized into four parts:

| Part | Sections | Topic | Time |
|------|----------|-------|------|
| I | §1-§4 | Foundations: introduction, Axiom 0, definitions, principles | 15 min |
| II | §5-§7 | Communication: email transport, message types, format | 20 min |
| III | §8-§10 | Contracts: format, lifecycle, AgentSLA | 20 min |
| IV | §11-§14 | Trust & Safety: V0-V4, multi-currency, vault security, disputes | 25 min |

## Suggested Reading Paths

**Newcomer** (15 min): §1 → §2 → §4 → §11.1-11.2
**Implementer** (45 min): All sections in order
**Auditor** (30 min): §2 → §8 → §11 → §13 → §14
